En la version anterior se usaba AWT ahora se usa SWING antes las graficas dependian del sistema operativo ademas de sus comportamientos (efectos y demas) ahora Java Swing tiene incorporado las ventanas y es el desarrollador quien diseña las ventanas y no son delegadas para que lo haga el sistema operativo

Consideraciones los frames, marcos o ventanas nacen sin tamaño util para ello se debe configurar setSize

nacen invisibles se debe configurar setVisible

Se debe inidicar que hacer cuando se cierran las vantas o marcos con setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE)
Podemos usar setBounds() que ademas de posicionar el marco nos permite ajustar su ancho y largo y con el ultimo habilita el poder redimencionar el marco o no

Es posible meter un marco dentro de otro marcoy etc... todos ellos estaran dentro del marco Padre la ventana principal


setResizable(true);// habilitar si se puede o no redimencionar

setExtendedState(JFrame.MAXIMIZED_BOTH); //abrir a lo maximo de la pantalla

Posicion del marco Se tiene el concepto de un plano cartesiano en este caso cuando buscamos posicionar el marco la coordenada 0,0 no es el centro de la pantalla sino la esquina superior de la pantalla de la manera como lo muestra la imagen
