Conceptos Basicos * Codigo Fuente: conjunto de instruciones que interpreta el computador atravez de un lenguaje de programacion
* Sintaxis: especifica el orden de las palabras y simbolos en las expresiones. Ejemplo x = 8 +2
* Semantica: expecifica el significado de las expresiones. En el anterior ejemplo es que me refiero a una suma
* Origin de la estrategia Dividir y conquistaras: es una vieja idea del emperador Julio Cesar que practicaba en sus peleas y en su manera de gobernar
* Dividir y conquistaras en campo computacional esta tecnica propone dividir un problema en varios sub problemas mas pequeños y resolver cada uno de manera independiente. la solucion de todos los sub problemas es combinada para finalmente obtener la solucion del problema original
Pensamiento Critico o Analisis Critico no es como el dicho comenzemos que en el camino vemos Definicion: se entiende como el proceso de evaluar y eligir a travez del razonamiento y la voluntad, una determinada opcion en medio de un universo de posibilidades, con el proposito de resolver una situcion concreta. Pilares:
1. curiosidad: Preguntar, es la opcion mas eficaz, Documentacion.
2. Razonamiento: Contexto es decir no fijarnos solo en los detalles: anilizar
3. Logica: desarrollo del sentido comun. hasta el grado de argumentar mi posicion
Cualidades 1. Calaridad: esponer sus ideas y para hace preguntas dando razonamientos claros y con argumento.
2. Contextualizacion: entiende el panorama general del problema y el problema Basico
3. Mente abierta: considera seriamente otros puntos de vista y esta alerta a escuchar alternativas
4. bien Informado: utilizando fuentes confiables y las referencia.
5. Paciente: espera tener sufiente evidencia antes de emitir juicios y cambia de posicion ante evidencias
Solucion de problemas * Especializacion: es una tecnica para utilitzar ejemplos particulares y especificos que abarquen el contexto del problema para brindarnos una idea cual puede ser el camino a la respueseta concreta y sus argumentos
* Generializar: al obtener resultados especificos son utiles y nos ayudan a buscar una respuesta general (patron, metodo, algotitmo).
Nota: para lograr desarrollar logica adecuada es util seguir el proceso: como se indica en la imagen de tres pasos

Metodo ideal para Solucion computacional Es el preceso de aboradar un Problema de manera sitematica para crear(codificar) una solucion que pueda ser realizada o ejecutada por un dispositivo computacional.
Nota: sistematico metodo de ordenacion, organizacion o clasificacion de elementos, es decir se estroctura una serie de tarias para elaborar el programa. ademas en el proceso internamente esta constituido por una secuencia de tareas en un orden determinado y con objectivos especificos que ayudan a construir el programa y su solucion

Ejemplo 		Identificar y convertirlo en una oportunidad para desarrollar una solucion creativa
Define la informacion que es util, lo que desean y necesitan los interesados. ademas de las tareas o funciones que llevan a la solucion
Estrategia es explorar alternativas de solucion mediante la realizacion de ejemplos de ejemplos particulares (Especializacion)
para plantear una estrategia general (generializacion).
Algotitmo(el como) se diseña la soluccion de forma detallada, especificando el conjunto de algotitmo para solucionar el problema planteado
Logro: se implementa la solucion en un lenguaje de Programacion y se valida el programa en un conjunto de pruebas
algotitmo es una secuencia finita de instruciones precisas(no ambiguas) que realiza una tarea especifica para solucionar un problema
Flujo de control: es el orden en que realiza las instruciones
Requirimiento o Requisito de software: define una funcion que el programa que se va construir bebe proporcionar para contribuir a solucionar un problema en un contexto real
Pseudocodigo es una especie de programa escrito en un lenguaje de programacion, que igual a español permite escribir ideas.
si bien la idea del Pseudocodigo es que sea mas flejible, hay que tener unas consideraciones al escribir un algotitmo en Pseudocodigo, al usar verbos en infinitivo (ingresar, leer, obtener) que serviran para expresar los algortimos
Diagramade flujo es la representacion visual de nuestros algortimos: que muestra los pasos en varias cuadros y muestra el proceso conectando los cuadros entre si Partes de un algortimo
* Input o entrada: es el ingreso de datos que necesita el algortimo para funcionar
* Proceso: se trata la operacion logica formal que el algortimo emprendera con lo recibido del imput
* Output o salida: los resultados obtenidos del proceso sobre el input, una vez terminada la ejecucion del algortimo
Que es Programar es una conjunto de instruciones escritas en un Lenguaje de programcion que realizan una o varias tares especificas cuando son ejecutadas por un dispositivo computacional
